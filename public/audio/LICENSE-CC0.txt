War World Synthesized Sound Pack — CC0
======================================
This CC0 dedication applies to the PROCEDURALLY SYNTHESIZED sound pack — the
output of `npm run sounds` (tools/gen-sounds.mjs), which uses no recordings and
no third-party samples. That synthesized pack is dedicated to the public domain
under CC0 1.0 Universal:
https://creativecommons.org/publicdomain/zero/1.0/
You may copy, modify, and use it for any purpose without attribution.

NOTE: the .wav files currently shipped in this directory are ElevenLabs-
generated, NOT CC0 — see LICENSE.txt. Run `npm run sounds` to replace them with
the CC0 synthesized pack.
